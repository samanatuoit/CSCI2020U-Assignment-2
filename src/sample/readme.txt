CSCI 2020U Assignment 2 - File Server

Installation:

1. Start IntelliJ
2. File -> Project From Version Control -> In GitHub repository URL paste: https://github.com/samanatuoit/CSCI2020U-Assignment-2.git
3. In Client.java in the getLocalFiles() method, change the value of the variable myDirectory to the directory
 that will contain the files that the client wishes to share.
4. In Server.java under the go() method, change the value of the variable serverDirectory to the directory you want
the server to share files on.
5. Run Server first in IntelliJ and then Client
6. Enjoy sharing text files between client and server
